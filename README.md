Цей застосунок надає користувачам можливість створювати, переглядати, оновлювати та видаляти завдання (CRUD-операції).
Користувачі можуть призначати завдання різним особам або командам. 
Також передбачено функціонал для встановлення пріоритетів (Низький, Середній, Високий) і термінів виконання, а також для відстеження статусу завдання (На черзі, В роботі, Завершено). 
Доступні функції фільтрації та пошуку для швидкого знаходження потрібних завдань.
